### js 中this的指向

1. 在普通函数和箭头函数的区别（详见箭头函数和普通函数的区别.md），普通函数的this指向调用对象，箭头函数本身不生成this，由外层的词法作用域决定this

2. 由new构造，指向其新创建的对象
```js
let A  = function() {
    this.a = a;
}
let o = new A(2);
console.log(o.a);//2
```
